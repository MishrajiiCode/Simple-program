# Assignment Submission -- Python Programs

This assignment contains two simple Python programs.\
The first one is a basic calculator that performs arithmetic operations,
and the second one is a program that prints a personalized greeting
using the user's full name.

------------------------------------------------------------------------

## Task 1: Basic Calculator

### What the program does:

-   Asks the user to enter two numbers.\
-   Performs **addition, subtraction, multiplication, and division**.\
-   Shows the result of each operation on the screen.

### Example:

    Enter the first number: 12
    Enter the second number: 4
    Addition: 16.0
    Subtraction: 8.0
    Multiplication: 48.0
    Division: 3.0

------------------------------------------------------------------------

## Task 2: Personalized Greeting

### What the program does:

-   Asks the user to enter their **first name** and **last name**.\
-   Joins them together to make a full name.\
-   Prints a **friendly greeting message** using the full name.

### Example:

    Enter your first name: John
    Enter your last name: Doe

    Hello, John Doe! Welcome to the Python program.
